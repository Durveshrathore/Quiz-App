# Quiz App

A responsive Quiz Application built using HTML, CSS, and JavaScript. Users can test their knowledge by selecting different categories, answering timed questions, and reviewing their performance at the end.

## Features

- Multiple quiz categories
- 15-second countdown timer
- Progress bar
- Real-time score tracking
- Instant answer feedback
- Detailed answer review
- Keyboard shortcuts (1–4 and Enter)
- Responsive and modern UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```
Quiz-App/
│── index.html
│── styles.css
│── script.js
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Durveshrathore/Quiz-App.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
